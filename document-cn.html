使用文档

属性名 说明 默认值
templateId table容器div的id table
url 如果设置此属性，uiduck将会请求网络数据资源。使用AJAX跨域请求。 无默认值
url 要求为String类型的参数，（默认为当前页地址）发送请求的地址。 无默认值
type 要求为String类型的参数，请求方式（post或get）默认为get。注意其他http请求方法，例如put和delete也可以使用，但仅部分浏览器支持。 无默认值
dataType 与ajax的dataType同义。 无默认值
data 要求为Object或String类型的参数，发送到服务器的数据。 无默认值
key 返回数据的key。如：list:[{xx:xx},{xx:xx}],将key设置为list即可 无默认值
limit 后端分页下返回数据的每页显示条目个数的key。 无默认值
page 后端分页下返回数据的当前页数的key。 无默认值
total 后端分页下返回数据总条目数的key。 无默认值
data 表格数据，格式为JSON数组，示例:
[{"name":"张三"},{"name":"李四"}]		 []
loading 如果设置此属性，uiduck将会显示加载动画 无默认值
icon 加载动画的图标，uiduck为您内置了10种方案。 无默认值
shade 遮罩层的透明度，如不需要可不设置此项。 无默认值
shadeColor 遮罩层颜色。 无默认值
blur 模糊的blur值，单位为像素，如不需要可不设置此项。 无默认值
time 加载动画的显示时间。 无默认值
style 表格样式设置 { trClass: "", thClass: "", tdClass: "" }
trClass 追加 tr 的样式， 无默认值
thClass 追加 th 的样式 无默认值
tdClass 追加 td 的样式 无默认值
topBar 顶部工具栏，可以放置搜索等相关工具，默认关闭 { templateId: "search", kwLight: true, kwSplite: false, kwSpliteList: [] },
templateId 指定topbar的div容器id，示例：<div id="topBar" uiduck-keyWord="name"></div"> 无默认值
kwLight 是否开启关键词高亮 false
kwSplite 是否分词高亮 false
kwSpliteList 分词高亮数组，如青岛XXX集团,后台分词后转为 ['青岛','XXX','集团']，通过分词高亮，将列中的青岛AAAXXXVV集团中的关键词青岛,XXX,集团高亮。 []
fieldOptions 表头和数据渲染 []
index 是否添加索引 false
key 表格数据列的唯一标识 无默认值
title 标题名称 无默认值
width 列宽,可设置为百分比和数字。数字下单位为像素 无默认值
style 追加的css样式，示例:bakground：red 无默认值
type 字段类型，如images,map,function 无默认值
function 当type为function时传入一个自定义的function方法，用于实现自定义计算 无默认值
colspan 于表头可横跨的列数 1
computed 计算绑定值，如computed: { "0": "李白", "1": "杜甫", "2": "白居易", "3": "王安石" }为1时显示杜甫 无默认值
hide 是否隐藏此列 false
page 是否开启分页 false
rightTool 右侧工具栏配置 rightTool: { templateId: "barDemo", title: "操作", width: "100px" },
templateId 工具栏容器div的id 无默认值
title 标题 无默认值
width 宽度 无默认值
language 多语言设置，如没有配置此项，将会根据浏览器的语言自动设置 language: { tag: "CN", options: [] },
tag 语言的标签，如：ChineseEnglish Chinese
options 字典配置 options = { "udHome": "首页", "udTotal": "共", "udList": "条", "udPrev": "上一页", "udNext": "下一页", "udLast": "尾页", "udGo": "前往", "udPage": "页", "udSub": "确定", "udNodata": "暂无数据" };

分页配置说明

支持前端后端、网络请求和本地数据分页。算法可参考源码

属性名 说明 默认值
autoNext 是否自动翻页 autoNext: { time: 2000, showPage: true }
time 间隔时间，单位毫秒 无默认值
showPage 自动翻页时是否显示分页栏 无默认值
pageOptions 分页配置 pageOptions: { pageClass: "", style: "", limit: "", dataType: "back", limits: ['5', '10', '15', '20', '25', '30'], layout: ["total", "home", "prev", "next", "last", "set", "jump"], index: 0, count: 0 }
pageClass 底部分页栏的样式 无默认值
emClass 底部分页栏当前页的样式 无默认值
limit 每页显示的条数 10
limits 每页条数的选择项。 ['5', '10', '15', '20', '25', '30']
layout 分页的排版，total 为总条数，home 为首页，prev 为上一页，next 为下一页，last 为尾页，set 为设置每页显示的条数，jump 为跳转到 ["total", "home", "prev", "next", "last", "set", "jump"]
index 当前页的索引 0
count 总页数 0

方法说明

以下均为对外方法。如需查看内置方法，可以参考源码。

方法名 说明 参数
new uiduck(String,{Object}) 初始化并创建uiduck对象。
var xxx = new uiduck('xxx',{...})
示例：var uiduck = new uiduck('uiduck1',{ templateId: "table", url: { url: "http://localhost/uiduck/getList", type: "GET" }})
uiduck.getRow(Object) 获取该行数据,一般用于右侧工具栏操作编辑或删除。 例： var xxx = new uiduck('xxx',{...})
xxx.getRow(this)
uiduck.skipPage(int) 跳转到第N页 例： var xxx = new uiduck('xxx',{...})
xxx.skipPage(N)
uiduck.refresh() 重新加载数据并渲染 例： var xxx = new uiduck('xxx',{...})
xxx.refresh()
uiduck.setData(Object,Boolean) 重置数据。如数据模式为网络数据，且不想使用内部继承的ajax,可以不使用uiduck.url配置。使用自定义的网络请求获取数据，并在通过此方法将数据置入。本地数据boolean设为false，网络数据设为true。 例： var xxx = new uiduck('xxx',{...})
xxx.setData([{"id":"1","name","张三"},{"id":"2","name","李四"}],false)
